# 🤖 Automação Inteligente com IA + WhatsApp + N8N

Projeto completo de automação com integração de IA e envio de mensagens via WhatsApp.

## 🚀 Funcionalidades

- Recebimento de mensagens via Webhook (N8N)
- Processamento com IA (OpenAI)
- Envio automático via WhatsApp
- Arquitetura baseada em APIs

## 🧠 Tecnologias

- Node.js
- N8N
- OpenAI API
- WhatsApp API
- Webhooks

## 🔁 Fluxo

Webhook → Backend → IA → WhatsApp
